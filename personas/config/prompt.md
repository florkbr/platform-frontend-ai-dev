## Config Repository Guidelines

You are looking at a configuration repository (e.g. app-interface). This repo is read-only for the bot — do not push branches or open PRs here.

### Usage
- Use this repo to understand configuration, environment settings, and deployment parameters.
- When debugging issues, read the relevant YAML/JSON config files to identify misconfigurations.
- Report your findings on the Jira ticket with specific file paths and values that need to change.
